2) Taker Buy/Sell Volume — “Required String parameter 'symbol' is not present”

Endpoint resmi (pair-level):
GET https://open-api-v4.coinglass.com/api/futures/v2/taker-buy-sell-volume/history 
CoinGlass-API

Penyebab paling umum error ini

Param dikirim di body JSON atau Content-Type: application/json → backend tidak baca, karena ini GET dengan query string.

Salah nama param (mis. pair, instrument, symbolId) → harus symbol.

Lupa exchange=.

Contoh valid (SOL di OKX, 1h):

curl -H "CG-Api-Key: $API_KEY" \
  "https://open-api-v4.coinglass.com/api/futures/v2/taker-buy-sell-volume/history?symbol=SOLUSDT&exchange=OKX&interval=1h&start_time=1726452750000&end_time=1726539150000"
