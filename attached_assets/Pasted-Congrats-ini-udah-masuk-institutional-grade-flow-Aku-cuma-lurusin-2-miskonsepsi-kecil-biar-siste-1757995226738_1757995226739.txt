Derivasi cepat:

TakerDominance = buy_usd / (buy_usd + sell_usd)

OI_Delta = OI_close - OI_open (H1)

OB_Imbalance = bids_usd / (bids_usd + asks_usd)

Trigger Long (5m):

TakerDominance > 0.55 dan OI_Delta > 0

Funding netral→positif ringan (hindari over-crowded)

OB_Imbalance > 0.52 atau harga reclaim OB/ FVG levelmu
→ kirim alert LONG, set TP di cluster asks terdekat, SL di bawah liquidity pocket terakhir.

Trigger Short (5m):

TakerDominance < 0.45 dan OI_Delta > 0 saat harga naik (potensi squeeze balik)

Funding positif tinggi (crowded longs)

OB_Imbalance < 0.48
→ alert SHORT, TP di cluster bids, invalidasi di atas wall absorbsi.
