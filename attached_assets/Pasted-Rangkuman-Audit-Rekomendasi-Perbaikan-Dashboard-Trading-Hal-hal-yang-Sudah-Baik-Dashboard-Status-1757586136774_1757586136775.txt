Prioritas Perbaikan (Action Plan)
Perbaiki Multi‑Coin Screening
Endpoint screener mengembalikan HTML sehingga parsing JSON gagal. Pastikan API mengirim JSON, pasang validasi content-type di front-end, dan tampilkan pesan error yang jelas.
Perbaiki Navigasi “OI Sentiment Analysis”
Tautan ini saat ini kembali ke dashboard utama. Periksa route/slug dan hubungkan ke komponen analisis sentimen.
Aktifkan dan Monitor WebSocket
Banyak halaman menunjukkan status WebSocket Disconnected. Pastikan URL WS dan token tepat, implementasi auto-reconnect, dan tampilkan indikator “Connected/Reconnecting” di UI.
Paket Kepercayaan Data
Tambahkan label sumber data (mis. OKX, CoinAPI) serta Last updated (WIB) dan latensi (ms) di setiap panel untuk meningkatkan kredibilitas. Gunakan format angka konsisten (pemisah ribuan, desimal sesuai range).
Error & Empty State
Buat tampilan skeleton loader dan pesan ramah saat data belum tersedia (“Sedang memuat…”). Jika terjadi error (401/429/500), sampaikan penyebab dan cara retry.
Observability & Logging
Implementasikan logging terstruktur (requestId, status, body length) agar lebih mudah menelusuri bug. Pasang monitoring untuk error rate, WS disconnect, dan response time.
Keamanan & Rate Limiting
Terapkan rate limit pada endpoint publik, sanitasi parameter input, dan tambahkan header keamanan (CSP, HSTS, X-Content-Type-Options).
Pengayaan & Notifikasi
Integrasi backtesting/performance tracking untuk memantau win‑rate sinyal, serta modul notifikasi ke Telegram/Discord (untuk alert breakout FVG, perubahan OI ekstrem, dll.).
UX Responsif
Pastikan tampilan responsif di perangkat mobile, sediakan tombol Refresh Data, dan tawarkan pilihan tema gelap/terang.
Roadmap Singkat
Minggu 1: Fix screener & routing OI Sentiment, tampilkan status WS+timestamp, implementasi error state.
Minggu 2: Stabilisasi WS dengan fallback polling, caching 3–10 detik, standardisasi format angka.
Minggu 3: Perbaiki security header, rate limiting, rilis halaman metodologi/disclaimer untuk transparansi data.
Kesimpulan
Dashboard sudah mengintegrasikan banyak fitur analisis tingkat institusi. Dengan perbaikan pada reliabilitas endpoint, pengelolaan WebSocket, transparansi data, dan UX error handling, platform ini akan terlihat lebih profesional, dipercaya, dan siap untuk pengguna besar.
